# Guía de Estilo y Diseño: Landing Presentación

Este documento define la coherencia visual y de contenido para la landing page de **Transcriptor IA**.

## 1. Concepto Visual
- **Temática**: Matrix / Hacker / Terminal Pro.
- **Paleta de Colores**:
  - `FONDO`: #0a0a0a (Negro profundo)
  - `ACENTO`: #55f198 (Verde Matrix brillante)
  - `ACENTO_DIM`: #2a7a4a (Verde apagado para bordes)
  - `TEXTO`: #e0e0e0 (Blanco roto/grisáceo)
- **Tipografía**:
  - `Mono`: Roboto Mono (Para datos, comandos y etiquetas)
  - `Serif`: Inria Serif (Para descripciones y narrativa elegante)

## 2. Tono de Comunicación
- **Sencillo**: Evitar términos técnicos como "FFmpeg", "PyInstaller" o "Backend".
- **Directo**: Enfocado en el beneficio ("Cualquier audio a texto") no en la tecnología.
- **Accionable**: Todo debe conducir al botón de descarga.

## 3. Secciones Clave
1. **Hero**: Promesa clara de transcripción universal.
2. **Descarga**: Botón destacado con metadatos simples (Windows, Gratis).
3. **Tutorial Visual**: 
   - Obtener API Key (Groq).
   - Instalación (Extraer y Ejecutar).
   - Uso (Subir y Transcribir).

## 4. Elementos de Diseño Reutilizables
- **Bordes**: 1px solid #1a3d1a.
- **Efectos**: Matrix Rain de fondo (opacidad 0.04), Scanlines sutiles.
- **Botones**: Bloques sólidos sin bordes redondeados.
- **Animaciones**: Revelado por scroll y efectos de "scramble" de texto.
